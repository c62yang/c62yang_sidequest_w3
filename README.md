## Description

This p5.js project is an interactive, multi-screen story game that uses a state-based structure to manage different scenes. The player progresses through a series of multiple-choice decisions that influence a tracked player stat called karma. Each decision branches the narrative slightly and contributes to the final outcome.

The game demonstrates how to separate logic across multiple files, manage game states, and create simple UI interactions using mouse input. Depending on the playerâ€™s choices, they will reach either a win or lose ending.
---

## Learning Goals

- Understand how to manage multiple game states using a shared state variable
- Practice separating code across multiple files for readability and organization
- Implement interactive multiple-choice gameplay using mouse input
- Track a player stat (karma) across scenes
- Create branching outcomes based on player decisions
- Design simple UI elements for instructions, choices, and endings

---

## How to Play

- Click anywhere on the screen to begin
- For each scene, choose between two options
- Click the left side of the screen for option A
- Click the right side of the screen for option B
- Each choice will increase or decrease your karma
- Your final karma determines whether you win or lose

---
## Game Structure

The game is divided into multiple states:

Start Screen
Displays instructions and game title

Game Scenes
Multiple decision points where player choices affect karma

Win Screen
Triggered if the player achieves high karma

Lose Screen
Triggered if the player has low karma
